# 📊 Excel Data Analysis Project 

## 📌 Overview
This project is an Excel-based data analysis system that combines **Sales Analysis** and **Employee  Management**. It helps to generate insights from raw data such as revenue trends, tax calculations, salary distribution, and workforce analytics.

---

## 🎯 Objectives
- Analyze medicine sales performance by category and time period  
- Track total revenue, tax, and discounted revenue  
- Evaluate employee salary structure and HR metrics  
- Generate meaningful business insights using Excel  

---

## 🧾 Datasets Included

### 💊 Sales Dataset
Contains:
- Order ID, Medicine, Category  
- Units Sold, Unit Price, Order Date  
- Total Revenue, Tax Amount, Discounted Revenue  
- Revenue Category, Month, Quarter  

### 👨‍💼 Employee Dataset
Contains:
- Employee ID, Name, Department  
- Salary, Join Date, Birth Date  
- Tenure, Age, Experience Level  
- Salary Grade, Bonus Amount, Status  

---

## ⚙️ Key Features

### 💊 Sales Analysis
- Total revenue calculation  
- Category-wise revenue analysis  
- Monthly performance tracking  
- Tax and discount calculation  
- Identification of highest and lowest orders  
- Revenue classification (Low / Medium / High)

### 👨‍💼 HR Analysis
- Total salary budget calculation  
- Average salary by department  
- Employee age and tenure analysis  
- Bonus distribution tracking  
- Staff segmentation (Senior / Junior / Mid-level)

---

## 📐 Excel Functions Used
- `SUM()` – Total calculations  
- `AVERAGE()` – Average values  
- `IF()` – Conditional logic (Revenue Category, Status)  
- `COUNTIF()` – Counting categories and employees  
- `MAX()` / `MIN()` – Highest and lowest values  
- `DATEDIF()` – Age and tenure calculation  
- `VLOOKUP()` – Data lookup (if used)
